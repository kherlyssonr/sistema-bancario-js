# 🏦 Sistema Bancário em JavaScript

Um sistema bancário desenvolvido em **JavaScript puro** como projeto de aprendizado contínuo.

O objetivo deste projeto é colocar em prática todos os conceitos estudados durante minha jornada como desenvolvedor Front-end. A cada novo conteúdo aprendido, novas funcionalidades serão adicionadas ao sistema, acompanhando minha evolução na linguagem.

---

## 🚀 Objetivo

Desenvolver um sistema bancário simples que evolua gradualmente conforme o aprendizado em JavaScript, aplicando conceitos de lógica de programação e boas práticas de organização de código.

Este projeto será atualizado constantemente durante meus estudos.

---

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📚 Conceitos aplicados

### ✅ Versão 1.0

* Variáveis
* Condicionais (`if` e `else`)
* Loops (`while`)
* Funções
* Operadores matemáticos
* Organização do código

---

## ✨ Funcionalidades atuais

* Consultar saldo
* Realizar depósitos
* Realizar saques
* Validação de saldo insuficiente
* Menu interativo
* Encerramento do sistema
